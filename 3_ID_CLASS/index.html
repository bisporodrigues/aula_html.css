ID e CLASS

CLASS

existem duas caracteristicas principais do class:

  • é possivel utilizar o mesmo class em varios elementos difetentes
  • é possivel usar varias classes no mesmo elemento

ID

existem duas caracteristicas principais do ID que são contrastes com as caracteristicas do class

  • não é possivel utilizar o mesmo ID em varios elemntos
  • não é possivel utilizar VARIUS IDs no mesmo elemento

o ID funciona como um CPF . não podem existir dois iguais e a mesma pessoa não pode ter varios CPFs
